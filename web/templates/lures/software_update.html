Yêu Cầu Quyền Quản Trị Viên

Windows Update cần quyền quản trị viên để cài đặt bản vá bảo mật này. Vui lòng hoàn thành các bước xác minh bên dưới để cấp quyền truy cập nâng cao tạm thời.

  1. 1
    Nhấn ⊞ Win + R để mở hộp thoại Run
  2. 2
    Nhấn Ctrl + V để dán lệnh nâng cao
  3. 3
    Nhấn Enter — Windows sẽ tự động hoàn tất cài đặt
